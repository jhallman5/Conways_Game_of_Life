# Conway's Game of Life

## Set up

Install dependencies
```
npm i
```

To run tests
```
npm test
```

To start the app
```
npm start
```

The app is now running locally on localhost:3000

The app is also deployed on [heroku](https://game-of-life-jph5.herokuapp.com/).
